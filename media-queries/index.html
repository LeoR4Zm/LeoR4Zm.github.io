CSS Media Queries

Code Log — How they work, why they’re useful, and four core media features

1) How does a CSS media query work, and why is it useful?

A CSS media query lets you apply styles only when certain conditions about the user’s device or environment are true. The basic form is @media (<condition>) followed by a block of CSS rules. When the condition matches—such as a viewport width under 700px, a dark color scheme, or a user who prefers reduced motion—the browser uses those rules. Media queries are the foundation of responsive design: they help the same HTML adapt to phones, tablets, laptops, large monitors, and accessibility preferences without duplicating pages. 

@media (max-width: 700px) {
  .grid { grid-template-columns: 1fr; }
}

In short, media queries let your layout, typography, spacing, and features feel “native” to each context—improving usability, performance, and accessibility.

2) Four media features (with explanations & live examples)

max-width — adapt to smaller viewports

max-width checks the viewport width and applies styles when it is at or below a certain value (e.g., 700px). It’s commonly used to switch from multi-column (desktop) to single-column (mobile) layouts. 

@media (max-width: 700px) {
  .demo-card { font-size: 1rem; padding: 1rem; }
}

orientation — portrait vs. landscape

orientation targets the current screen orientation. It’s useful for adjusting layouts on tablets/phones when the device rotates. 

@media (orientation: landscape) {
  .banner { min-height: 40vh; }
}

prefers-color-scheme — light/dark mode

Respects the user’s OS/browser theme preference. Provide a light default, then enhance for dark mode without JS. 

@media (prefers-color-scheme: dark) {
  :root {
    --bg: #0f1115;
    --fg: #e6e6e6;
    --card: #1a1f29;
    --accent: #6aa1ff;
  }
}

prefers-reduced-motion — accessibility for motion sensitivity

If a user indicates they prefer less motion (e.g., due to vestibular disorders), this feature lets you reduce or disable non-essential animations. 

@media (prefers-reduced-motion: reduce) {
  .spin { animation: none; }
}
