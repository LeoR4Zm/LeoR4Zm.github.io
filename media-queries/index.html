CSS Media Queries

Code Log — How they work, why they’re useful, and four core media features

1) How does a CSS media query work, and why is it useful?

A CSS media query lets you apply styles only when certain conditions about the user’s device or environment are true. The basic form is @media (<condition>) followed by a block of CSS rules. When the condition matches—such as a viewport width under 700px, a dark color scheme, or a user who prefers reduced motion—the browser uses those rules. Media queries are the foundation of responsive design: they help the same HTML adapt to phones, tablets, laptops, large monitors, and accessibility preferences without duplicating pages. 

@media (max-width: 700px) {
  .grid { grid-template-columns: 1fr; }
}

In short, media queries let your layout, typography, spacing, and features feel “native” to each context—improving usability, performance, and accessibility.
