CSS Color

This page explains how CSS uses color, including color, background-color, and border-color, and the main value formats (named, hex, RGB, and more). Author: Zimo Liu, for DTC355 activity14.

What do these CSS properties do?

color

Sets the foreground (text) color of an element. It can also affect currentColor usages (e.g., SVG strokes, borders using currentColor). 

p { color: #333333; }
a { color: rgb(0 102 204); }

background-color

Fills the background paint layer of an element’s box (content + padding by default). 

.note { background-color: lightgoldenrodyellow; }

border-color

Sets the color of an element’s border. Shorthand or per-side forms are allowed, and it respects currentColor if you want borders to track the text color. 

.card {
  border: 2px solid currentColor;
  color: #0a7; /* border will match this */
}

How does CSS accept color as a value?

Common, fully supported formats include named colors, hex, RGB/RGBA, and HSL/HSLA. Examples below:

Named

.named       { color: crimson; }
.named-bg    { background-color: cornflowerblue; }
.named-border{ border-color: tomato; }
Named color sample

Hexadecimal

.hex       { color: #1a1a1a; }
.hex-3     { color: #0af; }        /* shorthand */
.hex-alpha { background-color: #ff000080; } /* 50% alpha */
Hex color sample

RGB / RGBA

.rgb     { color: rgb(34 139 34); }          /* space syntax */
.rgba    { background-color: rgba(255, 165, 0, 0.25); } /* legacy comma syntax */
RGB/RGBA sample

HSL / HSLA (another format)

.hsl-bg     { background-color: hsl(200 80% 60%); }
.hsla-border{ border-color: hsla(340, 90%, 50%, 0.6); }
HSL/HSLA sample

Tip: currentColor is a special keyword equal to the computed color value, useful to keep borders/icons in sync with text.

Quick Accessibility Note

Ensure sufficient contrast between text (color) and background (background-color). A common heuristic is WCAG AA contrast ratio ≥ 4.5:1 for normal text.
